# MITRE ATT&CK Detection Mapping

## Failed Login Detection

Detection:
Repeated failed SSH login attempts.

MITRE ATT&CK:
- T1110 Brute Force

Severity:
Medium

---

## Brute Force Detection

Detection:
Multiple failed login attempts from a single IP.

MITRE ATT&CK:
- T1110.001 Password Guessing

Severity:
High

---

## Suspicious IP Detection

Detection:
Known malicious IP observed in logs.

MITRE ATT&CK:
- T1078 Valid Accounts

Severity:
High

---

## Privilege Escalation Detection

Detection:
User creation and elevated commands detected.

MITRE ATT&CK:
- T1068 Exploitation for Privilege Escalation

Severity:
High
