# Executive Summary

## Project

MITRE ATT&CK Mapping Lab

## Purpose

This project demonstrates how Security Operations Center (SOC) analysts classify detections and investigations using the MITRE ATT&CK Framework to understand attacker behavior and improve defensive capabilities.

---

## Key Capabilities

- MITRE ATT&CK Mapping
- Threat Classification
- Detection Engineering
- Threat Hunting
- Incident Investigation
- Security Reporting

---

## MITRE ATT&CK Coverage

| Technique | ATT&CK ID |
|-----------|-----------|
| Brute Force | T1110 |
| Password Guessing | T1110.001 |
| Valid Accounts | T1078 |
| Exploitation for Privilege Escalation | T1068 |

---

## Skills Demonstrated

- MITRE ATT&CK Framework
- Detection Engineering
- Incident Investigation
- Threat Hunting
- SOC Documentation
- Security Monitoring

---

## Outcome

This lab demonstrates practical MITRE ATT&CK mapping skills expected of SOC Analysts, Detection Engineers, Threat Hunters, and Incident Responders.
